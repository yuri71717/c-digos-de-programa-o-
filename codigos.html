Guia Completo de Códigos de Programação

HTML - Tags e Explicações

<!DOCTYPE>         - Define o tipo de documento como HTML5.
<html>             - Elemento raiz de um documento HTML.
<head>             - Contém metadados (como <meta>, <title>, <link>).
<meta>             - Define metadados como charset, viewport etc.
<title>            - Define o título da página (aparece na aba do navegador).
<body>             - Contém todo o conteúdo visível da página.
<h1> a <h6>        - Cabeçalhos de nível 1 ao 6.
<p>                - Parágrafo de texto.
<br>               - Quebra de linha.
<hr>               - Linha horizontal.
<a href="">        - Link clicável.
<img src="">       - Imagem.
<strong>, <em>     - Texto em negrito ou itálico.
<ul>, <ol>, <li>   - Lista não ordenada, ordenada e item de lista.
<table>, <tr>, <td>, <th> - Tabelas, linhas e colunas.
<form>, <input>    - Formulário e campos.
<button>           - Botão clicável.
<div>, <span>      - Contêineres genéricos.

HTML – Códigos extras

<meta name="viewport" content="width=device-width, initial-scale=1.0"> — para responsividade em dispositivos móveis
<link rel="stylesheet" href="estilo.css"> — importar CSS externo
<script src="script.js"></script> — importar JS externo
<button type="submit">Enviar</button> — botão de envio em formulário
<input type="checkbox" name="aceito" value="sim"> — checkbox
<input type="radio" name="sexo" value="masculino"> — botão radio
<label for="inputId">Texto</label> — associar texto a input
<canvas> — área para desenhos gráficos via JS
<details> e <summary> — criar conteúdo dobrável
<progress> — barra de progresso
<figure> e <figcaption> — imagens com legenda

CSS - Propriedades e Explicações

color              - Cor do texto.
background-color   - Cor de fundo.
font-size          - Tamanho da fonte.
font-family        - Tipo de fonte.
margin             - Espaço externo ao elemento.
padding            - Espaço interno ao elemento.
border             - Borda do elemento.
width, height      - Largura e altura.
display            - Define o comportamento (block, inline, flex etc).
position           - Define a posição (relative, absolute etc).
top, left, right   - Usado com position.
text-align         - Alinhamento do texto.
justify-content    - Alinha itens em flexbox.
transition         - Efeitos de transição (animações).
z-index            - Ordem de sobreposição.

CSS – Propriedades e conceitos extras

flex-grow, flex-shrink, flex-basis — controle detalhado do flexbox
grid-template-columns, grid-template-rows — definir colunas e linhas no grid
@media — regras para responsividade
box-sizing: border-box — inclui padding e border no cálculo da largura/altura
visibility: hidden — esconde o elemento, mas mantém o espaço
pointer-events: none — desabilita cliques no elemento
animation-name, animation-duration, animation-iteration-count — criar animações CSS
calc() — cálculos matemáticos no CSS (ex: width: calc(100% - 50px);)
var(--minha-cor) — usar variáveis CSS personalizadas

JavaScript - Comandos e Explicações

let, const, var    - Declaram variáveis (let e const são modernos).
if, else, switch   - Estruturas condicionais.
for, while         - Laços de repetição.
function           - Declara uma função.
=>                 - Arrow function (função moderna).
alert()            - Mostra um alerta.
console.log()      - Exibe no console do navegador.
document.getElementById() - Acessa elementos do HTML.
addEventListener() - Adiciona eventos (clique, teclado etc).
fetch()            - Realiza requisição HTTP (API).
JSON.stringify()   - Converte objeto em string JSON.
JSON.parse()       - Converte string JSON em objeto.

JavaScript – Códigos e APIs extras

setTimeout(function, ms) — executa função após delay
setInterval(function, ms) — executa função repetidamente
clearTimeout(id), clearInterval(id) — cancelar os timers
localStorage, sessionStorage — armazenar dados no navegador
class — criação de classes em JS moderno
Promise, async/await — lidar com operações assíncronas
try { } catch(e) { } — tratamento de erros
map(), filter(), reduce() — manipulação funcional de arrays
JSON.stringify(), JSON.parse() — converter objetos em JSON e vice-versa
API Canvas para desenhos gráficos
API Fetch para chamadas HTTP
Event bubbling e event delegation (delegação de eventos)

PHP - Funções e Explicações

echo               - Exibe conteúdo na tela.
var_dump()         - Mostra informações detalhadas da variável.
if, else, elseif   - Estruturas de decisão.
for, foreach       - Laços de repetição.
function nome()    - Declara uma função.
\$_GET, \$_POST      - Captura dados do formulário.
htmlspecialchars() - Evita ataques XSS ao imprimir HTML.
mysqli_connect()   - Conecta ao banco de dados MySQL.
mysqli_query()     - Executa consulta SQL.

PHP – Funções e conceitos avançados

session_start() — iniciar sessão
header("Location: pagina.php") — redirecionamento
explode(), implode() — manipular strings e arrays
trim() — remover espaços no início/fim da string
filter_var() — validar e sanitizar dados
password_hash(), password_verify() — segurança em senhas
date(), time() — manipulação de datas
try { } catch (Exception $e) { } — tratamento de exceções
PDO para banco de dados com prepared statements (mais seguro)
include_once, require_once — incluir arquivos evitando duplicidade
var_export() — imprimir valor para reutilização no código

Java - Comandos e Explicações

public, private    - Modificadores de acesso.
class              - Define uma classe.
main()             - Ponto de entrada do programa.
System.out.println() - Imprime no console.
int, String, boolean - Tipos de dados.
if, else, switch   - Condicionais.
for, while         - Repetições.
ArrayList<>        - Lista dinâmica.
try, catch         - Tratamento de exceções.
new                - Cria um novo objeto.
extends            - Herança de classes.

Java – Comandos e conceitos extras

this — referência ao objeto atual
super — acessar método ou construtor da superclasse
interface — contrato para classes implementarem métodos
abstract class — classe base incompleta
enum — enumerações (valores fixos)
final — variáveis, métodos ou classes que não podem ser sobrescritos
synchronized — controle de acesso em multithreading
lambda expressions — funções anônimas (Java 8+)
Streams API — processamento funcional de coleções
Generics — tipagem genérica em coleções
try-with-resources — gerenciamento automático de recursos
Reflection API — inspeção de classes em tempo de execução
